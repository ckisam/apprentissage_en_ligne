Projet d'apprentissage en ligne¶

Ce projet s'appuie sur un article de Elad Hazan, Adam Kalai, Satyen Kale et Amit Agarwal Logarithmic Regret Algorithms for Online Convex Optimization . Les auteurs y prsentent trois algorithmes permettant de faire de l'optimisation convexe en ligne, en garantissant sous certaines conditions un regret logarithmique la date $T$. Avec un fonction de perte convexe $f_t$ et une dcision $x_t$ la date $t$, le regret est dfini par: $$ Regret : R_T = \sum\limits_{t=1}^T f_t(x_t) - \underset{x \in K}{\text{min}} \sum\limits_{t=1}^T f_t(x)$$

Les trois algorithmes tudis dans cet article sont :

  • Online Gradient Descent
  • Online Newton Step
  • Exponentially Weighted Online Optimization

Définition d'une fonctionde perte¶

L'investisseur subit une perte logarithmique: $$\mathcal{L}(x_t) = -\log(x_t^Tc_t)$$.

Gradient : $$\nabla f_t(x_t) = \frac{-1}{x_t^T c_t} c_t < 0$$ Hessienne : $$ \nabla^2 f_t(x_t) = \frac{1}{(x_t^T c_t)^2}\left( c_{t,i}c_{t,j} \right)_{i,j}>0 $$

In [6]:
def Loss(x, c):
    return -math.log((x * c).sum())

def Gradient(x, c):
    scalarProd = (x * c).sum()
    return -c / scalarProd

On choisit de normer l'ensemble des placements $1$. Les dcisions possibles appartiennent donc au simplexe $K = \{x \in \mathbb{R}_+^{n} \text{ tq } x^Tu = 1\}$ o le vecteur $u$ n'est compos que de $1$. On dfinit la fonction de projection sur cet ensemble.

In [7]:
def projectionSimplexe(y):
    d = len(y)
    u = np.ones(d)
    x = cvx.Variable(d)
    obj = cvx.Minimize(cvx.sum_squares(x - y))
    constr = [x >= 0, u * x == 1]
    prob = cvx.Problem(obj, constr)
    prob.solve()
 
    return np.array(x.value).squeeze().tolist()

Calcul du regret¶

Pour valuer la performance de nos algorithmes, nous calculons chaque tape l'erreur logarithmique. Pour obtenir le regret, il est donc ncessaire d'ajouter le terme associ la meilleure slection de portefeuille sur l'ensemble des priodes : $\underset{x \in K}{\text{min}} \sum\limits_{t=1}^T f_t(x)$

On passe par un programme d'optimisation sous contrainte.

In [8]:
def objective(x, df):
    return df.apply(func = lambda y: Loss(x, y), axis = 1).sum()

def simplexConstraint(x):
    unitVect = np.ones(len(x))
    return (unitVect * x).sum() - 1

def bestPrediction(df):
    p = df.shape[1]
    startMin = np.ones(p)/p
    resOpt = scipy.optimize.minimize(fun = objective, x0 = startMin, args = (df),
                                  constraints = [
                                      {'type' : 'eq', 'fun' : simplexConstraint},
                                      {'type' : 'ineq', 'fun' : lambda x: x}
                                  ]).x
    res = dict()
    res['x'] = resOpt
    res['value'] = objective(resOpt, df)
    return res

1-Algorithme de la descente de gradient en ligne¶

Descente de gradient dans le cadre en ligne.

Pas : $\eta_1, ..., \eta_T$ avec $\eta_t=\frac{1}{H(t+1)}$

Etape 1: On choisit arbitrairement $x_1 \in K$

Etape 2: Pour tout $t>1$: on

$x_{t} = \Pi_K(x_t - \eta_{t+1} \nabla f_t (x_t))$

avec $\Pi_K(y)$: une projection sur $K$, $\Pi_K(y) = \underset{x \in K}{\text{argmin}} ||x-y||$

Avec l'hypothse de la forte convexit, la borne de regret est de : $$R_T \leq \frac{G^2}{2H}(1 + \log(T))$$

def onlineGradientDescent(df, init, H = 0.01):
    n = df.shape[0]
    start = time.time()
    print("--- ALGORITHME ONLINE NEWTON STEP")
    x_t_normalise = projectionSimplexe(init) 
    new_x_t = [ x_t_normalise ]
    loss = []
    prediction = []
    print("--- Nombre d'itrations : {}".format(n))
    it10pct = n // 10
    for i in range(df.shape[0]-1):
        c = df.iloc[i, ]
        eta_t = 1 / (H * (i+1))
        new_x_t += [projectionSimplexe((new_x_t[i] - eta_t * Gradient(new_x_t[i], c)).tolist())]
        c = df.iloc[i+1, ]
        loss += [ Loss(new_x_t[i+1], c) ]
        if i % it10pct == 0:
            print("----- {}% de l'algorithme".format((i // it10pct) * 10))
        
    print(" --- FIN DE L'ALGORITHME")
    print(" --- Temps coul : {} secondes".format(time.time() - start))
    res = dict()
    res["prediction"] = new_x_t
    res["loss"] = loss
    return(res)

2-Optimisation en ligne avec poids exponentiels¶

On commence par introduire une marche alatoire permettant d'chantillonner selon $w$.

Etape 1: Poids exponentiels $w_{t}(x) = e^{-\alpha\sum\limits_{\tau = 1}^{t} f_{\tau}(x)} = \prod\limits_{\tau = 1}^{t} h_{\tau}(x)$

Etape 2:Mise jour $x_{t} = \frac{\int_K x w_{t}(x)dx}{\int_K w_{t}(x)dx}$

Avec l'hypothse de la forte convexit, la borne de regret est de : $$R_T \leq \frac{n}{\alpha}(1 + \log(1 + T))$$

import random

def computeWeight(alpha, df, t, x):
    subDf = df.iloc[0:t,]
    prod = np.dot(subDf, x)
    res = math.exp(-alpha * prod.sum())
    return(res)

def randomWalk(alpha, df, t, delta):
    n = df.shape[1]
    x = np.ones(n) / n
    currentLkh = computeWeight(alpha, df, t, x)
    nbIt = 10 * df.shape[1]
    
    for i in range(nbIt):
        j = np.random.choice([k for k in range(n-1)])
        X = np.random.choice([-1,1])
        y = x.copy()
        y[j] = y[j] + X*delta
        y[n-1] = y[n-1] - X*delta
        newLkh = computeWeight(alpha, df, t, y)
        proba = min(1, currentLkh / newLkh)
        if np.random.binomial(1, proba) == 1 :
            x = y.copy()
            currentLkh = newLkh
    return x

def exponentiallyWeighted(df, init, alpha = 0.01, delta = 0.01):
    start = time.time()
    print("--- ALGORITHME EXPONENTIALLY WEIGHTED")
    n = df.shape[0]
    portfolio = [ init ]
    loss = []
    print("--- Nombre d'itrations : {}".format(n))
    it10pct = n // 10
    for i in range(n-1):
        portfolio += [ randomWalk(alpha, df, i, delta) ]
        loss += [ Loss(portfolio[i+1], df.iloc[i+1, ]) ]
        if i % it10pct == 0:
            print("----- {}% de l'algorithme".format((i // it10pct) * 10))
        
    print(" --- FIN DE L'ALGORITHME")
    print(" --- Temps coul : {} secondes".format(time.time() - start))
    res = dict()
    res["prediction"] = portfolio
    res["loss"] = loss
    return(res)

3-Algorithme de Newton en ligne¶

Etape 1: On choisit arbitrairement $x_1 \in K$

Etape 2: Pour tout $t>1$: on a

$$\Delta_{t-1}=\Delta f_{t-1}(x_{t-1})$$

$$A_{t-1}= \sum_{\tau=1}^{t-1}\Delta_{\tau}\Delta_{\tau}^T$$

$$b_{t-1}= \sum_{\tau=1}^{t-1}\Delta_{\tau}\Delta_{\tau}^T x_\tau- \frac{1}{\beta}\Delta_{\tau}$$

$$x_{t} = \Pi_K^{A_{t-1}}(A_{t-1}b_{t-1})$$

avec $\Pi_K^{A_{t-1}}(y)$: une projection sur $K$, $$\Pi_K^{A_{t-1}}(y) = \underset{x \in K}{\text{argmin}} (x-y)^T A_{t-1}(x-y)$$

Avec l'hypothse de la forte convexit, la borne de regret est de : $$R_T \leq 3(\frac{1}{\alpha}+4GD)n\log( T)$$

def projectionSelonMatrice(A, y) :
    d = len(y)
    u = np.ones(d)
    x = cvx.Variable(d)
    obj = cvx.Minimize(cvx.quad_form(x - y, A))
    constr = [x >= 0, u * x == 1]
    prob = cvx.Problem(obj, constr)
    prob.solve()
 
    return np.array(x.value).squeeze()

def onlineNewtonStep(df, init, beta = 0.01):
    start = time.time()
    print("--- ALGORITHME ONLINE NEWTON STEP")
    n, p = df.shape
    portfolio = [ init ]
    loss = []
    A = np.zeros((p, p))
    b = np.zeros(p)
    print("--- Nombre d'itrations : {}".format(n))
    it10pct = n // 10
    for i in range(n-1):
        c = df.iloc[i, ]
        delta = Gradient(portfolio[i], c).values.reshape(p, 1)
        Anew = np.dot(delta, np.transpose(delta))
        delta = np.transpose(delta)[0]
        bNew = np.dot(Anew, portfolio[i]) - delta / beta
        A += Anew
        b += bNew
        portfolio += [ projectionSelonMatrice(A, np.dot(np.linalg.pinv(A), b)) ]
        loss += [ Loss(portfolio[i+1], df.iloc[i+1, ]) ]
        if i % it10pct == 0:
            print("----- {}% de l'algorithme".format((i // it10pct) * 10))
        
    print(" --- FIN DE L'ALGORITHME")
    print(" --- Temps coul : {} secondes".format(time.time() - start))
    res = dict()
    res["prediction"] = portfolio
    res["loss"] = loss
    return(res)

Follow the approximate leader (FTAL)¶

Etape 1: On choisit arbitrairement $x_1 \in K$

Etape 2: Pour tout $t>1$: on

$$x_{t} = \underset{x \in K}{\text{argmin}} \sum_{\rho=1}^{t-1 } \tilde{f}_\tau (x)$$

o pour $\tau = 1 , ....,t-1$ on a $ \Delta_{\tau}=\Delta f_\tau (x)$ et

$ {f}_\tau (x) ={f}_\tau (x_\tau)+ \Delta_{\tau}^T (x-x_\tau) +\frac{ \beta}{2}(x-x_\tau)^T\Delta_{\tau}\Delta_{\tau}^T (x-x_\tau) $

def f_tilde(x, x_t, c, beta):
    scalarProd = (Gradient(x_t, c) * (x - x_t)).sum()
    return(Loss(x_t,c) + scalarProd + (beta / 2) * (scalarProd**2))

def f_tilde_sum (x, predList, df, beta):
    res = 0
    for i in range(len(predList)):
        c = df.iloc[i, ]
        res += f_tilde(x, predList[i], c, beta)
    return res

def simplexConstraint(x):
    unitVect = np.ones(len(x))
    return (unitVect * x).sum() - 1

def followApproximateLeader(df, init, beta = 0.01):
    start = time.time()
    print("--- ALGORITHME 'FOLLOW THE APPROXIMATE LEADER'")
    
    n, p = df.shape
    predList = [ init ]
    lossList = []
    
    print("--- Nombre d'itrations : {}".format(n))
    it10pct = n // 10
    
    for i in range(n-1):
        startMin = np.ones(p) / p
        newPred = scipy.optimize.minimize(fun = f_tilde_sum, x0 = startMin, args = (predList, df, beta),
                                         constraints = [
                                             {'type' : 'eq', 'fun' : simplexConstraint},
                                             {'type' : 'ineq', 'fun' : lambda x: x}
                                         ]).x
        predList += [ newPred ]
        c = df.iloc[i+1, ]
        lossList += [ Loss(newPred, c) ]
        
        if i % it10pct == 0:
            print("----- {}% de l'algorithme".format((i // it10pct) * 10))
        
    print(" --- FIN DE L'ALGORITHME")
    print(" --- Temps coul : {} secondes".format(time.time() - start))
    res = dict()
    res["prediction"] = portfolio
    res["loss"] = loss
    return(res)
